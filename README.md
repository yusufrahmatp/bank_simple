## Run Program (Ruby-only Version):

For the ruby only version, run
```
$ ruby ruby_only_ver/mechanics.rb
```
## Run Rails Program: 

1. Install rails

2. Run rails server
```
$ rails server
```
3. Access program at `http://local.host:3000`
